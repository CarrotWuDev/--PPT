# 智绘演示说明书

## 1. 软件概述

智绘演示是一款面向 AI 时代完整演示任务的智能演示生产软件。它不同于传统 PPT 工具以页面编辑为核心的工作方式，也不同于直接生成 PPT 产品以单次出稿为主要形态，而是围绕“生成—编辑—预览—演示—导出”的完整流程，构建一站式AI原生演示工作平台。

项目以 Markdown 作为结构化内容底座，结合 AI 参与内容生成、编辑优化和页面调整，并通过实时预览、演示控制和成果导出形成完整闭环。其目标不仅是提升 PPT 制作效率，更是探索 AI 时代演示内容生产与表达方式的新型工作流程。

## 2. 演示内容输入与准备

系统以 Markdown 作为演示内容的统一载体。用户既可以导入已有文稿，也可以直接新建内容，围绕演示主题组织标题层级、正文要点和备注信息，为后续页面生成与展示输出提供基础。

系统侧边栏支持大纲导航，可自动识别 Markdown 标题结构，帮助用户快速查看文稿框架并定位到对应内容进行编辑。

当用户尚未准备好完整文稿时，还可以通过 AI 输入主题或需求，快速生成初始 Markdown 草稿，提升演示内容准备效率。

在内容编辑过程中，用户既可以手动修改文本，也可以借助 AI 对 Markdown 内容进行辅助优化，如内容压缩、扩写、要点化整理和表达润色等，从而提升演示内容的条理性与表达效果。与此同时，系统侧边栏可自动识别标题结构并生成大纲，帮助用户快速定位和调整对应内容，提高文稿编辑效率。

## 3. 演示页面生成与预览优化

在完成演示内容输入与编辑后，系统能够根据 Markdown 文稿生成对应的演示页面，将结构化内容转化为可展示的页面结果，帮助用户快速形成演示初稿。

系统提供实时预览功能，用户可以直接浏览生成后的页面，检查展示效果，从而及时发现需要调整的问题。

![image-20260329114739560](C:/Users/31784/AppData/Roaming/Typora/typora-user-images/image-20260329114739560.png)

在预览过程中，用户还可以围绕当前页面中的局部元素发起 AI 辅助优化，对页面结果进行进一步完善，从而提升演示内容的表达效果和展示质量。

## 4. 演示展示与成果输出

在完成页面生成与预览优化后，系统支持用户直接进入演示展示环节。系统支持讲者视图与观众视图的协同展示，用户可根据实际汇报场景进行演示控制与内容呈现，从而更好地满足答辩、汇报和展示等任务需求。

![image-20260329152611177](C:/Users/31784/AppData/Roaming/Typora/typora-user-images/image-20260329152611177.png)

> 图4-1：讲者视图与观众视图的界面

在放映过程中，系统支持基本的演示流程控制与辅助功能（比如激光笔、计时器等），帮助用户更顺畅地完成现场展示，提升演示任务的完整性和实用性。

在成果输出方面，系统支持将当前演示结果导出为 PPTX 文件，方便用户在其他演示平台或办公环境中继续使用，提高演示成果的可迁移性和适用范围。
